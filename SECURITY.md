# Supported Versions

The table below shows the sable versions of Minecraft compatible with NextCommand.

* :heavy_check_mark: : Supported and tested
* :warning: : Not tested
* :x: : Not supported

| Minecraft Version    | Supported          |
| ---------------------| ------------------ |
| 1.16.3               | :heavy_check_mark: |
| 1.15.X               | :heavy_check_mark: |
| 1.14.X               | :warning:          |
| 1.13.X               | :warning:          |
| 1.12.X               | :warning:          |
| 1.11.X - 1.8.X       | :x:                |
| < 1.7.10             | :x:                |
